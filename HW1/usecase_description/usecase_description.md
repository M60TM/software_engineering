# step-by-step Use Case Description

## Use Case 분배
- 장의진: 설문 응답, 응답 설문 조회, 응답 취소, 응답 수정
- 천병선: 설문 검색, 설문 상세 정보 조회
- 김기현: 설문 등록, 설문 조회, 설문 삭제, 설문 통계
- 황재민: 회원 가입, 회원 탈퇴, 로그인, 로그아웃

---

## <황재민 파트>

### > 회원 가입
| Actor action | System response |
|---|---|
| 1. 사용자가 ID, 비밀번호, 이름, 전화번호, 이메일 등의 필수 입력 정보를 입력한다. |  |
|  | 2. 회원가입 성공 메시지를 출력하고 로그인 화면으로 이동 |

### > 회원 탈퇴
| Actor action | System response |
|---|---|
|  | 1. 안내문 출력 및 탈퇴 여부 질문 |
| 2-a. ‘탈퇴’ 버튼 클릭 |  |
| 2-b. ‘취소’ 버튼 클릭 |  |
|  | 3-a. 회원 탈퇴 메시지 출력 및 로그인 화면으로 이동 |
|  | 3-b. 이전 화면으로 이동 |

### > 로그인
| Actor action | System response |
|---|---|
|  | 1. 로그인 페이지 출력 |
| 2. 회원 또는 관리자가 ID와 비밀번호를 입력 |  |
|  | 3. 로그인 성공 메시지 출력 |

### > 로그아웃
| Actor action | System response |
|---|---|
| 1. 회원 또는 관리자가 로그아웃 버튼 클릭 |  |
|  | 2. 로그아웃 완료 메시지 출력 후 로그인 화면으로 전환 |

## <천병선 파트>

### > 설문 검색
| Actor action | System response |
|---|---|
| 1. 현재 진행 중인 설문 리스트, 향후 진행 예정인 설문 리스트, 종료된 설문 리스트 중 하나를 선택한 후 키워드 입력 |  |
|  | 2. 조건에 맞는 설문 리스트 출력 |

### > 설문 상세정보 조회
| Actor action | System response |
|---|---|
| 1. 검색된 설문 리스트 화면에서 특정 설문을 선택 |  |
|  | 2. 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각 등이 출력 |

## <장의진 파트>

### > 설문 응답

| Actor Action | System Response |
|---|---|
|  | 1. 설문 리스트를 출력한다. |
| 2. 응답 버튼을 누른다 |  |
|  | 3. 응답 화면으로 이동한다 |
| 4. 설문 작성 후 응답버튼을 눌러 제출한다 |  |
|  | 5. 설문 제출 메시지 출력 및 설문 상세정보 화면으로 이동 |

---

### > 응답 설문 조회(취소)

| Actor Action | System Response |
|---|---|
| 1. 응답 설문 조회 버튼 클릭 |  |
|  | 2. 응답한 모든 설문 정보가 출력되며, 설문 제목, 응답 내용, 응답 일시 등이 포함된다. |
| 2a. 현재 진행중인 설문 중, 응답 내용을 취소하고자 하는 설문의 취소 버튼을 누른다. |  |
|  | 2b. 설문 취소 메시지를 출력한다. |

---

### > 응답 설문 수정

| Actor Action | System Response |
|---|---|
|  | 1. 응답한 모든 설문 정보가 출력되며, 설문 제목, 응답 내용, 응답 일시 등이 포함된다. |
| 2. 현재 진행중인 설문 중, 수정하고자 하는 설문의 수정 버튼을 누른다. |  |
|  | 3. 새로운 설문 수정 화면을 띄우고, 가장 최근에 수정한 날짜가 출력된다. |
| 4. 설문을 수정하고 수정 버튼을 눌러 제출한다. |  |
|  | 5. 제출 완료 메시지 출력 후 응답한 설문 리스트 화면으로 이동한다. |
