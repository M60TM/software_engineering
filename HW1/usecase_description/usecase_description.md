# step-by-step Use Case Description

## Use Case 분배
- 장의진: 설문 응답, 응답 설문 조회, 응답 취소, 응답 수정
- 천병선: 설문 검색, 설문 상세 정보 조회
- 김기현: 설문 등록, 설문 조회, 설문 삭제, 설문 통계
- 황재민: 회원 가입, 회원 탈퇴, 로그인, 로그아웃

---
### 회원 가입
| Actor action | System response |
|---|---|
| 1. 사용자가 ID, 비밀번호, 이름, 전화번호, 이메일 등의 필수 입력 정보를 입력한다. |  |
|  | 2. 회원가입 성공 메시지를 출력하고 로그인 화면으로 이동 |

### 회원 탈퇴
| Actor action | System response |
|---|---|
|  | 1. 안내문 출력 및 탈퇴 여부 질문 |
| 2-a. ‘탈퇴’ 버튼 클릭 |  |
| 2-b. ‘취소’ 버튼 클릭 |  |
|  | 3-a. 회원 탈퇴 메시지 출력 및 로그인 화면으로 이동 |
|  | 3-b. 이전 화면으로 이동 |

### 로그인
| Actor action | System response |
|---|---|
|  | 1. 로그인 페이지 출력 |
| 2. 회원 또는 관리자가 ID와 비밀번호를 입력 |  |
|  | 3. 로그인 성공 메시지 출력 |

### 로그아웃
| Actor action | System response |
|---|---|
| 1. 회원 또는 관리자가 로그아웃 버튼 클릭 |  |
|  | 2. 로그아웃 완료 메시지 출력 후 로그인 화면으로 전환 |
